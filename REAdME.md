## ğŸŒŸ Overview

**SecureGen** is a lightweight, client-side utility designed to generate strong, secure passwords instantly. Built using **Vanilla JavaScript**, this tool emphasizes user privacy (no server-side processing) and ease of use.

1. **Set Length:** Enter a number between 4 and 32 in the "Password Length" input.
